Web development, or website development, are tasks associated with creating, building, and maintaining websites and web applications that run online on a browser.

Web development includes building technical front-end and/or back-end code that informs site functionality. Web developers may work exclusively on front-end code, back-end code, or both, and many modern jobs also require overlap in web design.

HTML or HyperText Markup Language is the language that allows you to create web application pages. Let's take the example of bold terms to attract the reader's attention. Some HTML Elements include semantic elements like header, main, section, and footer. There are many other elements like div, paragraph (p), ul or unorder list, etc.

CSS or Cascading Style Sheet allows to issue styles and allows to put the web pages into pages. For example, if you are designing content, you can change the size, font, color, division of the elements and other decorative elements. Some properties that are used in CSS are color, background color, padding, margin, etc.

Javascript allows the creation of the web page in order to improve the reciprocal influence of the reactors with the web page. For example, of a car, there are many different shapes, sizes, brands but the values are not the same. Some variables used in Javascript are const and let.

Javascript is one thing, HTML and CSS are another and to have a better understanding of JavaScript, developers often recommend learning HTML and CSS first.

HTML, CSS, and JavaScript all go hand and hand, and while you can do many things without knowing one, knowing all three can help you create beautiful, functional designs, applications and websites.
