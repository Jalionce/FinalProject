"Styles come and go. Good design is a language, not a style." -Massimo Vignelli

Web Development is a programming language that refers to the overall process to create web applications using coding. The 3 main focuses in Web Developement consists of the following:

  1. HTML - Hypertext Markup Language
  2. CSS - Cascading Style
  3. Javascript

Web Develop can be tedious like any other task that takes time, but there are many ways to make web development a joyful experience for anyone.

Our Vision

It's not easy trying to start developing websites. Helping beginner developers navigate through web development and providing insight will make things much easier.

Our Mission

No one thinks about how the internet works. By explaining the importance of IT and Web Development, we can create oppurtunities for ourselves and other junior developer to prove web development can be fun and easy.

Our Strategy

Thinking about it and doing it, two different things. Here, we provide resources and showcase our own skill to help better understand web development, implement creative ideas, and watch others enjoy IT.

“If there’s one thing you learn by working on a lot of different Web sites, it’s that almost any design idea–no matter how appallingly bad–can be made usable in the right circumstances, with enough effort.” ― Steve Krug
